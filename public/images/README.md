# Инструкция по добавлению логотипа

## Сохраните логотип GoldenHouse

1. **Сохраните изображение логотипа** (которое вы прислали) в эту папку:
   - Путь: `d:\Stroy\public\images\`
   - Название файла: **`logo-goldenhouse.png`**

2. **Убедитесь, что файл называется именно так:**
   ```
   logo-goldenhouse.png
   ```

## Альтернативный путь

Если у вас другое название файла или формат (например, `.svg`, `.jpg`), обновите импорты в:

- `src/components/layout/Header.tsx` (строка с `Image src="/images/..."`)
- `src/components/layout/Footer.tsx` (строка с `Image src="/images/..."`)
