# Логотипы брендов

Эта папка содержит логотипы брендов для страницы услуг по системам безопасности.

## Требуемые файлы:

### Видеонаблюдение:
- `hiwatch.png` (или .jpg, .jpeg, .svg)
- `hikvision.png`
- `dahua.png`
- `tantos.png`
- `reolink.png`
- `ezviz.png`
- `axis.png`

### Домофоны:
- `atis.png`
- `hikvision.png` (используется тот же файл, что и для видеонаблюдения)
- `eltis.png`

### Автоматика для ворот:
- `alutech.png`
- `doorhan.png`

### СКУД:
- `smartec.png`
- `parsec.png`
- `sigur.png`
- `rusguard.png`
- `bolid.png`

### Шлагбаумы:
- `nice.png`
- `faac.png`
- `came.png`
- `bft.png`
- `doorhan.png` (используется тот же файл, что и для автоматики ворот)

## Формат файлов:
- Поддерживаются форматы: PNG, JPG, JPEG, SVG
- Рекомендуемый размер: минимум 200x200px
- Фон: прозрачный (для PNG) или белый
- Имена файлов должны быть в нижнем регистре
