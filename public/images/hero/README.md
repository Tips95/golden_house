# Hero Background Images

Эта папка предназначена для фоновых изображений главной страницы.

## Требования к изображениям:

- **Формат**: JPG, PNG, WebP
- **Рекомендуемый размер**: 1920x1080px или больше (для Retina дисплеев - 3840x2160px)
- **Оптимальный размер файла**: до 500KB (рекомендуется сжатие)
- **Ориентация**: горизонтальная (landscape)

## Текущие файлы:

- `home-background.jpg` - основное фоновое изображение для главной страницы

## Как добавить новое изображение:

1. Поместите файл в эту папку
2. Обновите путь в `src/components/home/HomeHero.tsx`
3. Убедитесь, что изображение оптимизировано для веб-использования
