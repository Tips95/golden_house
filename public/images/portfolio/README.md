# Портфолио работ

Эта папка содержит фотографии выполненных работ.

## Для услуги "Подсветка лестниц и ниш"

Добавьте следующие изображения:

1. `lighting-stairs-1.jpg` - Фотография подсветки лестницы с датчиком движения
2. `lighting-niche-1.jpg` - Фотография подсветки ниши с RGB-управлением

### Рекомендации по изображениям:
- Формат: JPG или PNG
- Разрешение: минимум 1200x800px
- Размер файла: до 2MB
- Качество: высокое, хорошо освещенные фотографии
