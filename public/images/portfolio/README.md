# Портфолио по разделам (услугам)

Чтобы быстро добавить по 10–12 фото в каждый раздел:

## 1. Создайте папку по разделу

Имя папки = короткое имя раздела. Примеры:

| Папка        | Услуга / раздел              |
|-------------|-----------------------------|
| `plaster`   | Механизированная штукатурка |
| `facade`    | Фасадные работы             |
| `ppu`       | Утепление ППУ               |
| `elektrika` | Электромонтажные работы     |
| `burenie`   | Алмазное бурение и резка    |
| `podsvetka` | Подсветка лестниц и ниш     |
| `garazhnye` | Гаражные ворота             |
| `laminat`   | Укладка ламината            |
| `reklama`   | Наружная реклама            |
| …           | Полный список в `src/data/portfolio-folders.ts` |

## 2. Положите фото в папку

- Форматы: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
- Имена файлов любые (лучше латиница, без пробелов), порядок по алфавиту
- Рекомендуется: 10–12 фото на раздел, разрешение от 1200×800, до ~2 MB на файл

Пример структуры:

```
public/images/portfolio/
  plaster/     ← 10–12 фото штукатурки
  facade/      ← 10–12 фото фасадов
  ppu/         ← 10–12 фото ППУ
  elektrika/   ← 10–12 фото электрики
  …
```

## 3. Запустите синхронизацию

В корне проекта:

```bash
npm run portfolio:sync
```

Скрипт прочитает все папки в `portfolio/`, соберёт пути к фото и обновит данные. На странице услуги блок «Наши работы» подхватит фото автоматически.

## Поведение на сайте

- Если у услуги в коде уже задан массив `portfolioImages` — показывается он.
- Если нет — используются фото из папки (после `portfolio:sync`).
- Соответствие «slug услуги → папка» задаётся в `src/data/portfolio-folders.ts`.
