1. Зарегистрируйтесь на TimeWeb Cloud
   → https://timeweb.cloud/

2. Закажите VPS
   → Минимум: 1 CPU, 2 GB RAM, 10 GB SSD
   → Цена: от 300₽/месяц

3. Настройте DNS
   → A-запись: @ → IP_СЕРВЕРА
   → CNAME: www → ваш-домен.ru

4. Следуйте инструкции
   → START_HERE.md или DEPLOYMENT_QUICK.md


═══════════════════════════════════════════════════════════════
  ⚠️ ВАЖНО ПЕРЕД ДЕПЛОЕМ
═══════════════════════════════════════════════════════════════

1. Измените в .env.production:
   NEXT_PUBLIC_SITE_URL=https://ваш-домен.ru

2. Измените в nginx.conf:
   yourdomain.ru → ваш-реальный-домен.ru (2 места)

3. Загрузите проект в Git:
   git add .
   git commit -m "Ready for deployment"
   git push origin main

  💡 ПОЛЕЗНЫЕ КОМАНДЫ

Локально (проверка перед деплоем):
  npm install
  npm run build
  npm run start

На сервере (первый деплой):
  bash server-setup.sh
  git clone YOUR_REPO
  npm install && npm run build
  pm2 start ecosystem.config.js

На сервере (обновление):
  bash deploy.sh

Мониторинг:
  npm run status      # Статус PM2
  npm run logs        # Логи приложения
  npm run restart     # Перезапуск
